Here’s a Spring Boot rule of thumb
DTOs(A Data Transfer Object ) - Mostly public properties. Because their purpose is to transport data.
Dtos are to transfer data between application layers,
primarily between the REST Controller and the Service Layer

Rule of thumb
When you’re unsure, ask:
Is this object’s purpose to transport data, or to perform behavior?
Transport Data
data class UserResponse(...)
data class LoginRequest(...)
data class LoginResponse(...)
Usually: public val

Perform Behavior
class UserService(...)
class JwtService(...)
class PasswordValidator(...)
Usually: private val

DTO - public || Service - private
